# ShaderLab End-User Docs

This folder contains end-user documentation bundled with release builds and installer packages.

## Contents

- `INSTALL_AND_RUN.md` — install, first launch, and basic usage
- `KEYBOARD_SHORTCUTS.md` — common editor shortcuts
- `TROUBLESHOOTING.md` — common issues and fixes

## Where to Find These Docs in Release Artifacts

- Release build output: `build/bin/docs/enduser/`
- Installer install folder: `{app}\docs\enduser\`
- Portable fallback artifact: `docs/enduser/` inside the zip
